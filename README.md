# MEAN-seed

## What's in the Box?

1. Node
2. Angular
3. Mongoose (MongoDB ODM)
4. Jade (templating)

## Getting Started

1. ```$ git clone https://github.com/ajbraus/seed-mean.git seedMean```
2. ```$ cd seedMean```
3. ```$ npm install```
4. ```$ nodemon```
